# 小红书钩子公式库

## 六大钩子类型

### 1. 圈定人群
**公式：** 人群标签 + 悬念/利益点
- "想来浙江旅游的人注意了！"
- "25 岁还在用 XXX 的姐妹看过来"
- "第一次养猫的人千万别踩这个坑"
- "月薪 8K 想在北京租房的朋友，这条必看"

### 2. 直接提问
**公式：** 疑问词 + 反常识/好奇心
- "你知道杭州本地人周末都去哪吗？"
- "为什么你的咖啡在家冲永远不对味？"
- "同样的护肤品，为什么别人用了有效你用了没用？"

### 3. 自我否定
**公式：** 否定自身经验 + 转折
- "做了 5 年旅游博主，我居然不知道……"
- "用了三年 XX 产品，直到上周我才发现一直用错了"
- "以为自己很懂咖啡，直到被一个老师傅当面打脸"

### 4. 数据冲击
**公式：** 具体数字 + 颠覆认知
- "这个古镇 99% 的游客都走错了路"
- "测了 20 款面霜，真正值得买的只有这 3 款"
- "我花了 5000 块踩的坑，你 0 成本就能避开"

### 5. 场景代入
**公式：** 具体场景描绘 + 悬念
- "凌晨 5 点的西湖边，我看到了……"
- "打开快递箱的那一刻，我愣住了"
- "面试官问完最后一个问题，沉默了 10 秒"

### 6. 利益前置
**公式：** 直接说结果/好处
- "这个方法让我一个月瘦了 8 斤，不节食不运动"
- "用了这个插件之后，我每天多出 2 小时"
- "一招解决你 90% 的穿搭烦恼"

## 八大爆款元素

在选题时，检查是否命中至少一个爆款元素：

| 元素 | 核心情绪 | 示例 |
|------|---------|------|
| 成本 | 便宜又有面子 | "人均 30 的高级感下午茶" |
| 人群 | 身份认同 | "i 人专属周末计划" |
| 反差 | 意料之外 | "月薪 3000 vs 月薪 30000 的早餐" |
| 奇葩 | 猎奇心理 | "在鹤岗买了套房之后……" |
| 最差 | 避坑心理 | "千万别买的 5 款网红产品" |
| 怀旧 | 童年/青春 | "小时候校门口 5 毛钱的零食，你还记得几个？" |
| 头牌 | 好奇/仰望 | "全北京最贵的理发店，剪一次多少钱？" |
| 荷尔蒙 | 吸引力 | "被搭讪率最高的穿搭公式" |

## 选题公式

**选题 = 写作对象 + 人物关系 + 情绪立场**

示例：
- 写作对象：咖啡 + 人物关系：独居女生 + 情绪立场：治愈感 → "一个人住之后，我每天最期待的就是早上这杯咖啡"
- 写作对象：面试 + 人物关系：应届生 + 情绪立场：焦虑共鸣 → "投了 200 份简历之后，我终于想通了一件事"
