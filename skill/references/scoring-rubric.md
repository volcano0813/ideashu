# 内容质量评分标准

## 评分维度（每项 10 分，总分 60）

### 1. 钩子吸引力（前 3 行）

| 分数 | 标准 |
|------|------|
| 9-10 | 前 3 行有强悬念/强共鸣/强反差，读者不划走 |
| 7-8 | 有钩子但力度一般，勉强能留住人 |
| 5-6 | 钩子存在但太套路，读者可能划走 |
| 3-4 | 开头平淡，像在写作文 |
| 1-2 | 没有钩子，直接进入正文 |

**关键检查：** 前 3 行是否制造了"信息缺口"——让读者觉得不看完就亏了？

### 2. 个人真实感

| 分数 | 标准 |
|------|------|
| 9-10 | 有不可替代的个人细节（具体时间、地点、感受、数据），只有本人能写出来 |
| 7-8 | 有个人视角但细节不够具体，换个人也能编出来 |
| 5-6 | 有"我"的叙述但流于表面，像是编的 |
| 3-4 | 通篇第三人称或说教口吻 |
| 1-2 | 完全没有个人元素 |

**关键检查：** 如果把所有"我"替换成"某人"，内容是否毫无违和感？如果是，说明个人真实感不够。

### 3. AI 味指数（越低越好）

| 分数 | 标准 |
|------|------|
| 9-10 | 读起来完全像真人写的，有个人风格和不完美 |
| 7-8 | 基本自然，偶尔有 1-2 处 AI 感 |
| 5-6 | 有明显的 AI 痕迹但不严重（结构太工整/用词太书面）|
| 3-4 | AI 味较重（禁用词出现 3+ 次/结构完全对称/像"正确的废话"）|
| 1-2 | 一眼 AI（通篇禁用词/每段格式一样/没有任何口语化表达）|

**关键检查：** 参考 anti-ai-rules.md 逐项扫描。

### 4. 结构差异度

| 分数 | 标准 |
|------|------|
| 9-10 | 与同批次其他稿件结构完全不同（不同模板/不同叙事节奏）|
| 7-8 | 大结构不同但有部分段落节奏相似 |
| 5-6 | 结构有差异但能看出是同一个"模子"出来的 |
| 3-4 | 和其他稿件结构高度相似 |
| 1-2 | 几乎一模一样的结构，只是换了关键词 |

**关键检查：** 把这篇和同批次已生成的稿件放在一起，读者能否一眼看出是同一个人/系统写的？

### 5. 互动引导

| 分数 | 标准 |
|------|------|
| 9-10 | CTA 自然融入正文，读者有表达欲（想评论分享自己的经历）|
| 7-8 | 有互动引导但稍显刻意 |
| 5-6 | 结尾有"你们觉得呢"之类的套话 |
| 3-4 | 没有互动引导，内容自说自话 |
| 1-2 | 结尾生硬，读者没有回应的欲望 |

**关键检查：** 读完之后，你自己想不想在评论区说点什么？

### 6. 平台适配度

| 分数 | 标准 |
|------|------|
| 9-10 | 字数、emoji、段落、标签全部符合小红书规范 |
| 7-8 | 大部分符合，有 1-2 处小瑕疵 |
| 5-6 | 基本符合但不够精细（emoji 过多/段落过长）|
| 3-4 | 明显不适配（像公众号文章/知乎回答）|
| 1-2 | 完全不像小红书内容 |

**关键检查：** 参考 xiaohongshu-style.md 的规范逐项核对。

## 综合评判

| 总分 | 判定 | 建议动作 |
|------|------|---------|
| 50-60 | 优秀 | 直接进入定稿 |
| 42-49 | 达标 | 可进入定稿，建议微调标注的问题点 |
| 30-41 | 待改 | 返回阶段二，重点修改评分最低的 2 个维度 |
| < 30 | 重写 | 需要重新构思角度和结构 |
