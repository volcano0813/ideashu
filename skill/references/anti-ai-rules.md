# 去 AI 味规则

## 禁用词表

以下词汇和短语一旦出现，立即替换：

### 绝对禁用（出现即判定 AI 味过重）
- 毋庸置疑、不可否认、日新月异
- 总而言之、众所周知、综上所述
- 旨在、赋能、底层逻辑
- 值得一提的是、需要注意的是
- 不仅……更……、不仅……还……
- 与此同时、在此基础上
- 深入探讨、全面解析、一文读懂
- 干货、宝藏、绝绝子（已过气）
- 简直了、太绝了（过度使用时）

### 高危句式（容易触发 AI 味感知）
- "首先……其次……最后……" → 打乱顺序或用别的连接方式
- "一方面……另一方面……" → 用转折或并列的口语表达
- 每段都用 emoji 开头 → 随机分布，有的段不用
- 每个要点都是整齐的三行 → 长短不一，有的多说两句有的一句带过
- 结尾固定"你学会了吗？" → 换成更自然的收尾

## 结构变体

**不要每篇都用同一种结构。** 以下是可选的结构类型，每次批量生产时确保混合使用：

| 结构类型 | 特点 | 适合的内容 |
|---------|------|----------|
| 三段式 | 钩子→正文→CTA | 产品推荐、教程 |
| 对话体 | 模拟和朋友聊天 | 经验分享、种草 |
| 流水账体 | 按时间顺序记录 | 探店、旅行、测评 |
| 吐槽体 | 从吐槽切入再转推荐 | 避坑、对比 |
| 清单体 | 直接列条目 | 合集、清单、盘点 |
| 故事体 | 有起承转合的完整叙事 | 个人经历、成长 |
| 反转体 | 前半段铺垫后半段反转 | 测评、对比、打脸 |

## 自检清单

每篇内容生成后，逐项检查：

- [ ] 是否包含禁用词表中的任何词汇？
- [ ] 是否每段都整齐对称（长度、格式都一样）？→ 故意打破对称
- [ ] 是否通篇没有一句口语化的表达？→ 加入至少 2 处口语
- [ ] 是否没有任何"不完美"的地方（跑题、感慨、题外话）？→ 适当加入 1 处"题外话"让它更像真人写的
- [ ] 是否能看出明显的"AI 总结"痕迹（过于全面、面面俱到、滴水不漏）？→ 故意省略一些角度，留白
- [ ] 标题是否像 AI 取的（过于工整、用了"一文读懂""全面解析"等）？→ 换成更口语的标题
- [ ] 全文读起来是否像"正确的废话"——每句都对但没有信息增量？→ 加入具体的数字、场景、细节
